AsteriskTable HOC component for formatting nested data to a tree view. Takes flat data with parent id defined in key from parent_id_key prop when flat prop is true. Additionally, parent object can be defined in parent_key to speedup recursive expanding. Takes nested data with children listed in key from children_key prop when flat prop is false.
